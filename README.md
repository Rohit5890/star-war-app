# Star-war-app
Simple User Login and filter/ search planet app

# Getting Started
Follow below commands in sequence to get started
```
git clone <repo url>

npm i

npm start

Goto https://localhost:9000 to access app on your browser locally

```
# Prerequisites
 **`node` setup is must prior to above steps**
